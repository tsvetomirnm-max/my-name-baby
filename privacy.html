Поверителност и бисквитки

Какво събираме: псевдонимно устройство ID, оценки (1–5), пол/възрастова група ако са избрани, и по желание имейл при контакт. Не изискваме профил.

Защо: за да предотвратим дублирани гласове и да позволим да променяш своя вот от същото устройство.

Бисквитки: използваме една функционална бисквитка (fp_mnb) за устройствено ID. Ако откажеш бисквитки, запазваме ID само в паметта на устройството (localStorage) и приложението продължава да работи.

Право да бъдеш забравен: пиши ни през страницата „За нас“ и включи имейл/детайли, за да можем да намерим записа и да го изтрием.

Имейл & съобщения

Съобщенията от формата ни се съхраняват в база данни с цел обработка на обратна връзка.

GDPR/DSAR

За достъп/изтриване на данни свързани с твоето устройство ID или имейл, пиши ни през формата за контакт.

SQL за таблица „messages“ (пусни в Supabase → SQL)

create extension if not exists pgcrypto;
create table if not exists public.messages (
  id uuid primary key default gen_random_uuid(),
  created_at timestamp with time zone default now(),
  email text,
  body text not null,
  user_fingerprint text,
  user_agent text
);
